# Contributing

This repository is maintained as a public portfolio-safe prototype.

## Development Workflow

1. Install dependencies with `npm install`.
2. Run `npm run dev` for local development.
3. Run `npm run check` before opening a pull request.

## Code Guidelines

- Keep source changes focused and easy to review.
- Do not add real credentials, private endpoints, or production secrets.
- Prefer reusable components and route-level pages that match the existing structure.
- Keep demo auth and AI surfaces clearly separated from production claims.

## Pull Request Checklist

- Lint passes.
- TypeScript check passes.
- Production build passes.
- No secrets or private project names are included.
