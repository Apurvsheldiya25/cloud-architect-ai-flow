# Cloud Architect AI Flow

Public React/TypeScript prototype for an AI-assisted cloud architecture product site.

## What This Shows

- Product landing page for a cloud architecture workflow.
- Feature, pricing, roadmap, contact, login, and signup screens.
- Multi-cloud positioning across AWS, Azure, and Google Cloud.
- Reusable UI components built with shadcn/ui, Tailwind CSS, and Lucide icons.
- Clean Vite setup suitable for fast iteration and static deployment.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Run Locally

```sh
npm install
npm run dev
```

## Notes

This repository is a public portfolio-safe prototype. Private product code, internal project names, and production credentials are intentionally not included.
